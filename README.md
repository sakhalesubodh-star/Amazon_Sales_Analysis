#  Amazon Sales Analysis

# Project Overview
This project analyzes Amazon sales data to identify sales trends, top-performing products, and business insights using Python, Excel, SQL, and Power BI.

# Objectives
- Clean and preprocess sales data.
- Analyze sales performance using SQL and Python.
- Create an interactive Power BI dashboard.
- Generate business insights to support decision-making.

#Tools & Technologies
